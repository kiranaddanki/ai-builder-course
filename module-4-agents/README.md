# Module 4: Building AI Agents

## Overview
This module explores autonomous AI agents — systems that can reason, plan, and take multi-step actions to complete complex goals. You'll learn to design and build reliable agents using modern frameworks and best practices.

## Topics Covered
- What makes an AI agent: reasoning, memory, tools, and action loops
- Agent architectures: ReAct, Plan-and-Execute, multi-agent systems
- Building agents with the Claude Agent SDK and other frameworks
- Giving agents tools: web search, code execution, API calls, file access
- Managing agent memory: short-term context vs. long-term storage
- Human-in-the-loop patterns and agent safety
- Debugging and evaluating agent behavior
