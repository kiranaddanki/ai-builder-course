# Module 5: Building ML-Powered Bots

## Overview
This module focuses on building bots that combine traditional machine learning with modern LLMs. You'll learn to train, fine-tune, and deploy ML models alongside conversational AI to create powerful, data-driven bots.

## Topics Covered
- ML fundamentals refresher: classification, regression, clustering
- Training custom ML models for domain-specific tasks
- Fine-tuning LLMs on your own data
- Building chatbots and automation bots powered by ML models
- Combining rule-based logic with ML predictions
- Deploying ML bots to production (APIs, messaging platforms, etc.)
- Monitoring model drift and retraining strategies
